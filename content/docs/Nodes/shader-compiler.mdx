Unreal Engine has several issues with stuttering due to the shader compiler. This feature allows you to compile shaders in-game, making the experience more seamless.

## How to use

<Steps>
    <Step>Drag and drop `ABP_ShadersCompiler` to your menu level</Step>
    <Step>
        Get reference to the actor and them call `start` function from
        `ABP_ShadersCompiler`
    </Step>
</Steps>

-   Warning: Avoid compiling shaders in editor mode, as this will compile the engineâ€™s built-in materials and may lead to increased RAM usage.

## Nodes

### Start

-   Start shader compilation

<UFunction
    name="Start"
    className="ABP Shaders Compiler"
    pure={false}
    inPins={[
        { name: "", pintype: "exec" },
        { name: "Target", pintype: "object" },
    ]}
    outPins={[{ name: "", pintype: "exec" }]}
/>

### Stop

-   Stop shader compilation

<UFunction
    name="Stop"
    className="ABP Shaders Compiler"
    pure={false}

## Events

### On Started

-   Event triggered when shader start to compile

<UEvents
    name="On Started"

### On Shader Compiled

-   Event triggered when one shader has been compiled.
