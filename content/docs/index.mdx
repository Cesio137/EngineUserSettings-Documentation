This product is a plugin for Unreal Engine that allows you to customize various settings and features of the engine in game. You can change the graphics API, enable or disable ray tracing, nanite, global illumination, shadows, reflections, anti-aliasing, and other post-process effects. You can also switch between different shader models, feature levels and compile shaders in game to optimize your game performance. This plugin gives you more control and flexibility over the visual quality and performance of your Unreal Engine projects.

## Supported Unreal Versions

UE4          | UE5
-------------|----------------
4.26 - 4.27  | 5.1 - 5.5
